This project is aimed at providing webhook backend for Telegram bot where user can see information about queue size and estimated time to cross the border for different border checkpoints. User is encouraged to click two buttons - I started crossing the border and I'm through. System calculates time it took to cross the border as time interval between
those two clicks, and provides this info to other users. In case this info is not available, it provides estimated time to cross the border based on Telegram chats and current queue size, calculated by another project.
